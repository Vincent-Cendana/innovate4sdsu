## 🎯 Project Overview
Finding a seat in the library or a quiet corner in a cafe can be a challenge. **AztecSpaces** acts as a community database, similar to Yelp for study spots, where students can:
* **Search** for study locations based on name.
* **Rate** locations based on overall quality and noise/busyness levels.
* **Contribute** feedback via comments to help peers decide where to set up for their next study session.

---

## 🚀 Key Features
* **Interactive Search:** Quickly filter through campus study spots. 🔍
* **Rating System:** Visual 5-star ratings for Overall and Busy status. ⭐
* **Real-time Feedback:** Share your experiences with specific spots. 💬
* **Persistent Storage:** Comments are saved locally to your browser so your notes are always accessible. 💾
* **User Profiles:** Simple username based session tracking. 👤

---

## 🛠 Tech Stack
* **Frontend:** React.js
* **Routing:** React Router DOM
* **Storage:** Browser LocalStorage for lightweight data persistence
* **Styling:** Custom CSS-in-JS

## 📦 How to Run the Project
1. **Clone the repository:**
```bash
   git clone <your-repo-url>
```
2. **Install dependencies**
```bash
   npm install
```
3. **Start the development server:**
```bash
   npm start
```
4. **Open http://localhost:3000 in your browser.**
